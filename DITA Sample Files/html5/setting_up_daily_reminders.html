Setting Up Daily Reminders

How to set daily reminders on BreakBuddy.

These steps show you how to set up the frequency of your daily reminders for BreakBuddy. The app sends you reminders based on your selected schedule.

  1. Open the Break Schedule tab from the main dashboard.
  2. Select the Start Time and End Time for your daily breaks.
  3. Chose the Frequency of reminders (every hour, every 90 minutes).
  4. Tap Save Schedule.
